🏕️ Pre-Entrega - Tienda de Camping

Proyecto web desarrollado como pre-entrega para práctica de HTML5 y CSS3.
La página simula una tienda de productos para camping y asados, incluyendo navegación, tarjetas de productos, diseño responsive y formulario de contacto.

✨ Características
Diseño responsive con Media Queries
Tarjetas de productos reutilizables
Navegación interna con scroll suave
Formulario de contacto funcional con Formspree
Integración de Google Maps
Efectos hover y estilos modernos
Organización del CSS en múltiples archivos
🛠️ Tecnologías utilizadas
HTML5
CSS3
Flexbox
Media Queries
Google Fonts
📂 Estructura del proyecto
├── index.html
├── style.css
├── tarjetas.css
├── media_queries.css
├── tarjeta.html
└── imagenes/
